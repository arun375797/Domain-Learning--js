# Full Exercise List

Quick reference to every practice prompt, grouped by topic and listed with a
very short reminder of the goal.

- **Warmups**
  - `add` – sum two numbers.
  - `isEven` – check for even integers.
  - `swapValues` – swap `[a, b]` into `[b, a]`.
  - `celsiusToFahrenheit` – convert Celsius to Fahrenheit.
  - `calculateBMI` – compute BMI to one decimal place.
  - `difference` – absolute difference between two numbers.
  - `arrowSum` – rest-parameter sum with an arrow function.
  - `reverseArrayInPlace` – reverse an array without `.reverse`.
  - `countdown` – list numbers from `start` down to `0`.
  - `printMultiples` – first `times` multiples of `base`.

- **Control Flow**
  - `gradeFromScore` – return a letter grade from 0–100.
  - `maxOfThree` – largest value among three numbers.
  - `fizzBuzz` – classic FizzBuzz array up to `limit`.
  - `countVowels` – count vowels in a string, case insensitive.
  - `daysInMonth` – days in a month, including leap years.
  - `compareNumbers` – report greater/less/equal.
  - `sumUsingLoop` – compare sums via different loop styles.
  - `twoSumBruteForce` – all pairs summing to `target`.
  - `printEvenFromTwenty` – even countdown from 20 to 2.
  - `fizzBuzzObject` – FizzBuzz results keyed by number.

- **Functions**
  - `greet` – greet with a default name.
  - `sumAll` – sum any number of arguments.
  - `createMultiplier` – build a multiplier factory.
  - `once` – run a function once and cache the result.
  - `repeat` – invoke a function multiple times.
  - `memoize` – memoise via `Map`.
  - `curryAdd` – curried addition chain returning total.
  - `defer` – delay a function call by `ms`.
  - `compose` – compose functions right-to-left.
  - `withThis` – call with custom `this` binding.

- **Arrays**
  - `doubleValues` – double every number in a list.
  - `filterAdults` – keep users aged 18+.
  - `sumOdd` – sum only the odd numbers.
  - `findMax` – maximum value via `reduce`.
  - `flatten` – flatten one nesting level.
  - `findSecondLargest` – second-largest distinct number.
  - `removeDuplicates` – dedupe while preserving order.
  - `twoSumClosestToZero` – pair closest to zero sum.
  - `rotateRight` – rotate array right by `steps`.
  - `sumPositive` – sum only positive numbers.
  - `frequencyCounter` – map numbers to counts.
  - `removeNth` – remove index without mutation.
  - `pairWithTarget` – all unique pairs meeting `target`.
  - `moveZerosToEnd` – stable zero shuffling to end.
  - `chunk` – split into fixed-length chunks.

- **Objects**
  - `getFullName` – combine first and last names.
  - `pluckValues` – extract property values from items.
  - `mergePreferences` – overlay overrides on defaults.
  - `countOccurrences` – word occurrence tally.
  - `groupBy` – group by property value.
  - `isEmptyObject` – detect empty plain objects.
  - `highestValueKey` – key with highest numeric value.
  - `lowestValueKey` – key with lowest numeric value.
  - `removeOddValuedEntries` – drop odd-numbered values.
  - `deepClone` – deep-copy without `structuredClone`.
  - `freezeNested` – recursively `Object.freeze`.
  - `sumObjectValues` – sum numeric values (nested aware).
  - `mergeUniqueUsers` – merge arrays deduping by key.
  - `methodBorrowingExample` – borrow a method via call/apply/bind.

- **Strings**
  - `reverseString` – reverse characters.
  - `isPalindrome` – palindrome check ignoring case/spaces.
  - `capitalizeWords` – title-case each word.
  - `countCharacters` – character frequency map.
  - `maskEmail` – obfuscate local part of an email.
  - `capitalizeLastLetter` – uppercase final char per word.
  - `wordFrequency` – word count, case insensitive.
  - `swapWords` – reverse word order.
  - `reverseEachWord` – reverse characters in each word.
  - `isAnagram` – compare anagrams ignoring spaces/case.
  - `removeCharacter` – remove specified character instances.
  - `vowelCount` – count vowels in a sentence.
  - `formatTime` – format dates as `DD/MM/YYYY HH:mm`.

- **DOM**
  - `getListItems` – read text from `#todo-list` items.
  - `toggleHighlight` – toggle a `.highlight` class on click.
  - `appendTodo` – append `li` items to `#todo-list`.
  - `renderUserCard` – build a `.card` from user info.
  - `wireForm` – handle form submit and reset input.
  - `toggleCheckbox` – invert checkbox via button.
  - `preventRightClick` – disable context menu with message.
  - `attachDelegatedHandler` – event delegation helper.
  - `countdownTimer` – run and display a countdown.
  - `debouncedInput` – debounce input before callback.

- **Asynchronous**
  - `wait` – resolve after `ms`.
  - `timeout` – reject when another promise exceeds `ms`.
  - `fetchJson` – fetch and parse JSON.
  - `getUserNames` – fetch users and return their names.
  - `retry` – retry an async function up to `attempts`.
  - `printNumbersWithDelay` – sequentially log numbers with delay.
  - `sequentialTasks` – run async tasks one after another.
  - `parallelLimit` – run async tasks with concurrency limit.
  - `memoizedFetch` – fetch cache keyed by URL.
  - `raceWithFallback` – resolve with whichever promise finishes first.

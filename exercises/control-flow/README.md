# Control Flow Questions

Practice `if/else`, `switch`, and looping logic.

### Core Functions
1. `gradeFromScore(score)` — return `"A"`/`"B"`/... based on numeric ranges.
2. `maxOfThree(a, b, c)` — return the largest number.
3. `fizzBuzz(limit)` — return an array of values from 1..limit with Fizz/Buzz rules.
4. `countVowels(text)` — count vowels (a, e, i, o, u) in a string.
5. `daysInMonth(month, year)` — return number of days accounting for leap years.

### More Drills
- `compareNumbers(a, b)` — return `"greater"`, `"less"`, or `"equal"`.
- `sumUsingLoop(list)` — sum array values with `for`, `while`, and `forEach` to compare patterns.
- `twoSumBruteForce(numbers, target)` — find all pairs that add up to `target` using nested loops.
- `printEvenFromTwenty()` — loop from 20 down to 2 printing only even numbers.
- `fizzBuzzObject(limit)` — return an object keyed by number with `"fizz"`, `"buzz"`, `"fizzbuzz"`, or `"number"`.

# Warmup Questions

Quick drills to get comfortable with syntax and fundamental operations. Each
exercise lives in `exercises/<name>.js`, while `practice.js` re-exports the same
functions if you prefer a single entry point.

### Essentials
1. `add(a, b)` - return the sum of two numbers.
2. `isEven(number)` - return `true` if the number is even.
3. `swapValues(pair)` - swap two numbers contained in an array `[a, b]`.
4. `celsiusToFahrenheit(tempC)` - convert Celsius to Fahrenheit.
5. `calculateBMI(weightKg, heightMeters)` - return BMI rounded to one decimal.

### Extra Warmups from Pending List
- `difference(a, b)` - absolute difference of two numbers.
- `arrowSum(...numbers)` - rest-parameter arrow function that sums all inputs.
- `reverseArrayInPlace(list)` - practice swapping items manually.
- `countdown(start)` - return a countdown from `start` down to `0`.
- `printMultiples(base, times)` - generate the first `times` multiples of `base`.

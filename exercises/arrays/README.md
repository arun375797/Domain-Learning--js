# Array Practice

Strengthen iteration and transformation skills. Implement the prompts inside
`practice.js`.

### Core Set
1. `doubleValues(numbers)` — return a new array with each value doubled.
2. `filterAdults(users)` — given `{ name, age }` objects, keep age >= 18.
3. `sumOdd(numbers)` — sum only the odd numbers.
4. `findMax(numbers)` — return the largest number using reduce.
5. `flatten(nested)` — flatten a one-level nested array (`[1, [2, 3]] -> [1, 2, 3]`).

### Additional Challenges from Pending List
- `findSecondLargest(numbers)` — compute the second highest number without sorting.
- `removeDuplicates(numbers)` — drop duplicate entries (preserve order).
- `twoSumClosestToZero(numbers)` — return two numbers whose sum is closest to zero.
- `rotateRight(numbers, steps)` — rotate array elements to the right without using built-ins.
- `sumPositive(numbers)` — reduce the array to the sum of positive values only.
- `frequencyCounter(numbers)` — return an object counting occurrences of each number.
- `removeNth(numbers, index)` — remove the `index`th element without mutating the original.
- `pairWithTarget(numbers, target)` — list all unique pairs equaling `target`.
- `moveZerosToEnd(numbers)` — relocate zeros to the end while keeping order.
- `chunk(numbers, size)` — split into equally sized arrays (last chunk may be smaller).

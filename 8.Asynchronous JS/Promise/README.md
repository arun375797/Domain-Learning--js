# Promises

- `promise-basics.js` — create new Promise, resolve/reject.
- `chaining.js` — chain `.then` calls and return values.
- `combinators.js` — `Promise.all`, `Promise.allSettled`, `Promise.race`, `Promise.any`.
- `error-handling.js` — catch errors and use `finally`.

Always return promises from inside `.then` to maintain the chain.

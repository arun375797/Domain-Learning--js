# String Practice

Sharpen text processing techniques. Implement each challenge in `practice.js`.

### Core Problems
1. `reverseString(text)` — return the text reversed.
2. `isPalindrome(text)` — ignore case and spaces to detect palindromes.
3. `capitalizeWords(sentence)` — capitalize the first letter of each word.
4. `countCharacters(text)` — return a frequency map of characters.
5. `maskEmail(email)` — replace the middle of the local part with `*`.

### Extra Challenges from Pending Bucket
- `capitalizeLastLetter(sentence)` — title-case the last character of every word.
- `wordFrequency(sentence)` — count word occurrences regardless of casing.
- `swapWords(sentence)` — reorder `"hello world"` to `"world hello"`.
- `reverseEachWord(sentence)` — reverse characters in each word individually.
- `isAnagram(a, b)` — detect if two strings contain the same letters.
- `removeCharacter(text, char)` — strip all occurrences of `char` without regex.
- `vowelCount(sentence)` — return the number of vowels using iterative & regex approaches.
- `formatTime(timestamp)` — convert `Date` to readable `DD/MM/YYYY HH:mm`.

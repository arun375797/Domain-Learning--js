## Getting Started

1. Start with `00-setup/` to configure your environment
2. Follow the numbered sections sequentially (00-15)
3. Complete exercises in `exercises/` after each topic
4. Build projects in `projects/` to apply knowledge
5. Track progress in `resources/progress-log.md`

## Learning Path

See `resources/roadmap.md` for detailed learning paths and milestones.

## Organization

Each main topic is broken down into logical subtopics:
- **Basics** - Fundamental concepts
- **Advanced** - Complex patterns and techniques
- **Patterns** - Common design patterns and best practices

All files use consistent kebab-case naming for easy navigation.

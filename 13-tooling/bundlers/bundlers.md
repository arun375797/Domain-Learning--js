# Bundlers

- Vite: fast dev server, minimal config. `npm create vite@latest`.
- Webpack: highly configurable; set up entry, output, loaders, plugins.
- Rollup: tree-shaking friendly for libraries; configure input/output formats.
- Parcel/ESBuild: zero-config options for prototypes.
- Document how to add Babel or TypeScript to each bundler.

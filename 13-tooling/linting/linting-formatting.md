# Linting & Formatting

- ESLint: init with `npx eslint --init`, configure rules in `.eslintrc`.
- Prettier: install, add `"format": "prettier --write \"**/*.{js,ts,json,css,md}\""` to scripts.
- Integrate with editors via extensions and `.vscode/settings.json`.
- Husky + lint-staged: run checks before commits.

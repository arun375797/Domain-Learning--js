# Transpilation

- Babel: configure `.babelrc` or `babel.config.js`, install presets (`@babel/preset-env`).
- TypeScript: `tsconfig.json` basics (`target`, `module`, `strict`).
- SWC and esbuild: fast alternatives for transforming code.
- Polyfills: use core-js or regenerator-runtime; prefer browserslist to target environments.

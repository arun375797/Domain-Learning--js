# Environment Setup

- Install the latest LTS version of Node.js and verify with `node --version`.
- Add a code editor (VS Code recommended) and install the ESLint + Prettier
  extensions.
- Configure auto-format on save and enable `"editor.formatOnSave": true`.
- Initialize a project folder with `npm init -y` when you need package scripts.
- Keep a global `npx` shortcut list handy (`npx create-vite`, `npx eslint --init`).

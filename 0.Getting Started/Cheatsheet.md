# Quick Cheatsheet

- Restart Node REPL: press `Ctrl+C` twice.
- Clear console: `console.clear()` or `Ctrl+L` in DevTools.
- Pretty-print objects: `console.dir(obj, { depth: null })`.
- Format code: `npx prettier --write "**/*.js"`.
- Jump to definition in VS Code: `F12`; peek definition: `Alt+F12`.

# Daily Workflow

- Run quick experiments with the browser console or `node` REPL (`node` then `.`exit).
- Execute files via `node path/to/script.js` and add `"type": "module"` when using ES modules.
- Use `npm run lint` / `npm run format` scripts to keep code style consistent.
- Store useful snippets or links in `Cheatsheet.md` so you can reference them later.
